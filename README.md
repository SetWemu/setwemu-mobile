# SetWemu - Event Discovery & Ticketing Platform for Sri Lanka

Your go-to app for discovering local events, buying tickets, and connecting with your community.

## Project Structure
- `src/screens/` - All app screens
- `src/components/` - Reusable components
- `src/services/` - API and external services
- `src/navigation/` - Navigation configuration
- `src/assets/` - Images, icons, fonts
- `src/utils/` - Helper functions
- `src/constants/` - App constants and theme
